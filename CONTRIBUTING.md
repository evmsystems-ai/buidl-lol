# Contributing to buidl.lol

Thanks for your interest in contributing! 🙌

## How to Contribute

### 1. Fork the Repository

Click the "Fork" button on [github.com/evmsystems-ai/buidl-lol](https://github.com/evmsystems-ai/buidl-lol) to create your own copy.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/buidl-lol.git
cd buidl-lol
```

### 3. Set Up Upstream Remote

```bash
git remote add upstream https://github.com/evmsystems-ai/buidl-lol.git
```

### 4. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 5. Make Your Changes

- Follow the existing code style
- Add tests if applicable
- Update documentation as needed

### 6. Commit Your Changes

```bash
git add .
git commit -m "feat: add your feature description"
```

We use [Conventional Commits](https://www.conventionalcommits.org/):
- `feat:` — New feature
- `fix:` — Bug fix
- `docs:` — Documentation
- `chore:` — Maintenance

### 7. Push to Your Fork

```bash
git push origin feature/your-feature-name
```

### 8. Open a Pull Request

Go to [github.com/evmsystems-ai/buidl-lol](https://github.com/evmsystems-ai/buidl-lol) and click "New Pull Request".

## Development Setup

```bash
cd web
npm install
npm run dev
```

## Code of Conduct

Be respectful. We're all here to build cool stuff together.

## Questions?

Open an issue or join our [Discord](https://discord.gg/buidl).
